# NumPy Day 3

Statistics:

np.mean()     → Average
np.median()   → Middle value
np.std()      → Standard deviation
np.max()      → Maximum value
np.min()      → Minimum value

Logical Operations:

arr > value
arr < value
arr >= value
arr <= value
arr == value

Boolean Indexing:

arr[arr > value]

Counting Conditions:

np.sum(condition)
