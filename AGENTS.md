# AGENTS.md

## Repo Snapshot
This repository is a personal data science learning roadmap. Keep changes aligned with the existing month/week/day structure and the educational style used in the exercises.

## Working Rules
- Prefer small, local edits that preserve the current learning flow.
- Keep code and explanations beginner-friendly unless the file clearly targets a later topic.
- Do not rewrite commented-out practice code unless the user asks for cleanup or a fix.
- Use the existing folder pattern when adding new material: month -> week -> day -> practice/challenge/mini-project/notes.
- Treat notebook files and CSV/TXT data files as part of the learning content, not as generated build artifacts.
- Keep notes concise and bullet-oriented, matching the existing learning notes style.

## Useful References
- [README.md](README.md) for the high-level roadmap.

## What To Avoid
- Avoid introducing production-style architecture, heavy tooling, or unrelated abstractions.
- Avoid broad rewrites across multiple lessons when a narrow fix or addition is enough.
